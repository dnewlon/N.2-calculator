# MidAtlantic Energy - Nitrogen Calculator

A professional nitrogen (N₂) calculation tool for oil & gas field operations.

## Features

- **Well Foam Operations** - Calculate N₂ requirements for well unloading with foam
- **Pipeline Foam Pig** - Determine N₂ needed for foam pig pipeline operations  
- **Tank/Pipe Drying** - Calculate purge requirements for moisture removal
- **General N₂ Volume** - Convert between SCF and liquid gallons at any conditions

## Technical Details

- Real gas Z-factor calculations using Hall-Yarborough correlation
- Accurate across wide pressure (0-10,000 psi) and temperature ranges
- All calculations performed client-side (no data sent to servers)
- Mobile-responsive design

## Usage

Simply open `index.html` in any modern web browser, or host on any web server.
